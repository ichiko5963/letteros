## 💡 使い方のコツ

### 経験談の深掘り

抽象的な記憶 → 具体的な情景描写に変換するのがポイント:

**NG例**:
> 「昔、メルマガ作成に苦労していました」

**OK例**:
> 「2年前の秋、深夜2時。自宅の書斎で、ノートパソコンを開いている。
> 画面の光が目を刺し、スクロールしても情報が見つからない。
> 『あれだけ時間をかけたのに...』心の中でつぶやく。
> 焦りと絶望感で、呼吸が浅くなった。」

**変換のコツ**:
- いつ・どこで・何をしていた?
- 何が見えた・聞こえた・感じた?
- どんな感情?
- 心の中で何を考えていた?

### 数字の活用

具体的な数字を最低3つ入れる:

- **金額**: 「300万円」「年商1億」
- **人数**: 「33名」「720名」
- **期間**: 「3ヶ月」「2年」
- **その他**: 「90%」「10倍」

### 一貫したメッセージ

シーケンス全体で同じコアメッセージを伝える:

**例: 「上流設計が全てを変える」というメッセージの場合**

- 1通目: 上流設計がないと何が起こるか(問題)
- 2通目: 上流設計との出会い(転機)
- 3通目: 上流設計で変わった未来(成功)

## 📊 品質基準

### 文字数
- **最低**: 3000文字
- **推奨**: 3500〜5000文字
- **経験談**: 800〜1500文字

### 具体性
- **数字**: 3つ以上
- **固有名詞**: 2つ以上
- **五感の描写**: 2つ以上

### 構成
- **冒頭**: 具体的な情景描写
- **本文**: 経験談 + 問題/解決/成功
- **締め**: CTA + 追伸

## ❓ よくある質問

### Q1: 3000文字も書けない...
**A**: AIとの対話で経験談を深掘りしましょう。800文字の経験談が2つあれば、1600文字。残り1400文字は問題提起や解決策で埋まります。

### Q2: 数字が3つもない
**A**: 以下を振り返ってみてください:
- 期間(何ヶ月、何年)
- 金額(投資額、売上、損失)
- 人数(クライアント数、参加者数)
- パーセンテージ(改善率、達成率)
- 回数(何回試したか)

### Q3: 経験談が抽象的になってしまう
**A**: AIの5つの質問に答えましょう:
1. いつ、どこで?
2. 何をしていた?
3. 何が見えた、聞こえた、感じた?
4. どんな感情?
5. 心の中で何を考えていた?

### Q4: どのパターンを選べばいい?
**A**: 商品価格を基準に:
- 〜3万円 → 1通
- 3〜10万円 → 2通
- 10〜30万円 → 3通(推奨)
- 30〜100万円 → 4通
- 100万円〜 → 5通

### Q5: テンプレートをそのまま使っていい?
**A**: ❌ ダメです。テンプレートは「構造」と「要素」の参考にしてください。内容は必ず自分の経験と言葉で書きましょう。
